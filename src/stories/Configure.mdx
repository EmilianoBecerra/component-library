# Component Library

Component library for viewing components on demand.

## Components

- **TaxonomyTree** — Interactive tree that renders the entire hierarchy of a graph.
- **ChildrenLevel** — internal recursive component that handles each level of the tree.

## Tech

- React + TypeScript
- shadcn/ui (Collapsible, ScrollArea)
- Tailwind CSS
- Storybook
